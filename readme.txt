
Программа получает на вход путь (FILEPATH), ведущий к папке, в которой находятся файлы с текстами в формате *.txt 
(для корректной работы программы считаем, что иных объектов, в том числе файла с кодом самой программы, в папке не находится).
Путь указывается полностью, например: FILEPATH = 'C:/Users/user/PycharmProjects/курсач/main3_acc/'
Алгоритм алгоритма размечает недостающие после обработки автоматическим акцентуатором ударения 
в соответствие с существующими в русской литературной традиции вариантами метрической схемы гекзаметра.
Каждый размеченный текст записывается в новый файл.
Запускается программа через:
if __name__ == '__main__':
    process_all_files(FILEPATH, 3)
	
Рекомендуемым интерпретатор: PyCharm
